题目:在我国，“教育”一词最早起源于( )中的“得天下英才而教育之，三乐也”。
类型:单选题
A.《论语》
B.《礼记》
C.《孟子》
D.《说文解字》
正确答案:C
解析:【 解 析 】在我国，一般认为“教育”一词最早见于《孟子 ·尽心上》中的“得天下 英才而教育之， 三乐也”。许慎最早在《说文解字》中对“教育”一词进行解释。 易错提示“ 教育”一词在我国的最早出处(使用)与最早解释是易混点，考生应注意区分。最 早出处：孟子的《孟子 ·尽心上》———“得天下英才而教育之，三乐也”。 最早解释：许慎的《说文解字》——“教，上所施，下所效也”“育，养子使作善也”。

题目:教育活动的基本构成要素包括教育者、受教育者和（ )（易混）
类型:单选题
A.教育影响
B.教育手段
C.教育物资
D.教育活动方式
正确答案:A
解析:【解 析】由题干及选项设置可知，本题考查的是三要素说。教育者、受教育者(学习者) 和教育 影响(教育媒介)是构成教育活动的基本要素。 方法技巧 关于教育活动的基本要素，不同的学者有不同的观点。以下归纳了几种常考的说 法，考生做题时应注意具体问题具体分析。 三要素说：教育者、受教育者、教育影响/教育媒介/教育措施/教育内容。 四要素说：教育者、受教育者、教育内容、教育手段/教育活动方式/教育物资。

题目:下列属于教育家孟禄关于教育起源的观点的是（ )
类型:单选题
A.生物的冲动是教育的主要动力
B.教育起源于儿童对成人的无意识模仿
C.教育起源于生产劳动过程中经验的传递
D.教育是在生活中提出的，也是在生活中进行的
正确答案:B
解析:孟禄持心理起源说，他认为教育起源于日常生活中儿童对成人的无意识模仿。 A 项 属 于生物起源说的观点，C 项属于劳动起源说的观点，D 项属于生活起源说的观点。

题目:教育的本质属性是( )
类型:单选题
A.传授知识
B.传承文化
C.培养人才
D.服务社会
正确答案:C
解析:教育的本质属性是育人，即教育是一种有目的地培养人(培养人才)的社会活动。 故 选 C 项。 方法技巧 理解教育的本质属性需要重点抓住“培养人”三个字。(1)“培养人”说明教育活 动不 同于人类的其他社会活动：①它是以人为直接对象的，不同于以物质产品或精神产品 的生产为直 接对象的社会生产活动；②它是以对人的身心发展产生影响为直接目标的，不 同于医疗活动以及 以满足人的需要为目标的社会服务活动。(2)“培养人”确认了教育的存 在特性。教育活动可以 加速人的社会化品质的塑造，通过培养人服务于社会。(3)“培养人” 是教育的主体活动。

题目:从个体的角度来定义“教育”，往往把“教育”等同于个体的( ）
类型:单选题
A.社会化过程
B.学习与发展过程
C.人格化过程
D.认知过程
正确答案:B
解析:一般说来，人们是从两个角度给“教育”下定义的：一个是社会的角度，另一 个是个体 的角度。从个体的角度来定义“教育”,往往把“教育”等同于个体学习与发展的 过程。兼顾社会 和个体两个方面给教育下定义：教育是在一定社会背景下发生的促使个体 的社会化和社会的个性化的实践活动。A 项排除，C、D 项为干扰选项。

题目:教育发展要走在经济发展的前面，这表明教育具有（ ）
类型:单选题
A.依附性
B.永恒性
C.时代性
D.相对独立性
正确答案:D
解析:教育的社会属性的具体内容如下： 社会属性 内涵 永恒性 教育是人类所特有的社会现象，与人类社会共始终 历史性 不同时期的教育有不同的历史形态和特征 继承性 不同时期的教育有共同特点，前后相继 长期性 无论从一个教育活动完成角度，还是从一个个体的教育生长角度，其时间周 期都比较长 相对独立性 教育有其自身的规律，具有相对独立性，可以“超前”或“滞后”于当时的 社会发展 生产性 教育是生产性活动，与其他生产活动相比，在对象、过程与结果等方面有自 己的特性 民族性 教育在具体的民族或国家中进行，有其民族性特征 教育发展要走在经济发展的前面也就是教育先行或教育超前发展，体现了教育的相对独立性。

题目:教育活动中，师生共同认识的客体要素是（ ）
类型:单选题
A.教育环境
B.教育手段
C.教育内容
D.教育方法
正确答案:C
解析:从微观角度看，教育活动由教育者、受教育者、教育内容和教育手段四个要素 构成 其中，教育内容是教育活动中师生共同认识的客体，教育手段是教育活动的基本条件。 C 项正确。

题目:“教育起源于儿童对成人的‘无意识模仿’”，这一观点抹杀了教育的（ )
类型:单选题
A.遗传性
B.自然性
C.目的性
D.获得性
正确答案:C

题目:下列关于教育的社会属性，表述正确的是( )
类型:单选题
A.“十年树木，百年树人”表明教育具有历史性
B.教育发展会受到政治经济的影响，因此教育与政治经济同步发展
C.教育是培养和发展人的活动，所以在一定意义上具有生产性
D.教育既受当时生产力的制约，同时也受生产关系的制约，体现了教育具有继承性
正确答案:C
解析:教育作为一种培养和发展人的活动，具有生产性的特点，与其他生产活动相比， 在对 象、过程与结果等方面有自己的特殊性。“十年树木，百年树人”表明 教育具有长期性的特点。教育的相对独立性表明教育受一定社会的生产力发 展水平和政治、 经济制度制约、决定，但与社会生产力发展水平和政治经济制度的改变并 非完全同步，具有与社 会发展的不平衡性。教育的历史性表明在不同的社会 或同一社会的不同历史阶 段，教育的性质、目的、内容等各不相同。这是因为教育既受当 时生产力的制约，同时也受当时生 产关系的制约。

题目:人的教育活动与动物的“教育”活动的本质区别是( )
类型:多选题
A.模仿性
B.意识性
C.规律性
D.社会性
正确答案:B,D
解析:教育是人类社会特有的活动，动物界不存在教育。社会性和意识性是人的教育 活动 和动物的“教育”活动的本质区别。动物界的某些行为虽与人类社会的教育相类似， 但本质不同。

题目:教育活动由“教”和“学”构成，两者相互依存。因此，教育的主体性要素包括( )
类型:多选题
A.教育者
B.教学场所
C.教学媒体
D.受教育者
正确答案:A,D

题目:教育的生物起源学说认为( ）
类型:多选题
A.教育是只有人才有的本能
B.教育是动物的本能
C.教育是种族发展的需要
D.教育不仅存在于人类社会，而且存在于动物界
正确答案:B,C,D
解析:教育的生物起源学说的代表人物是法国社会学家、哲学家利托尔诺与英国教 育家 沛西 · 能。利托尔诺在《各人种的教育演化》一书中认为，教育活动不仅存在于人 类社会之中，而 且也存在于人类社会之外，甚至存在于动物界；不仅在脊椎动物中存在， 甚至在非脊椎动物中也存 在。人类社会的教育是对动物界教育的继承、改善和发展。沛西·能在 1923 年不列颠协会 教育科学组大会上以“人民的教育”为题说，“教育从它的起源来说，是一个生物学的过程”, “生物的冲 动是教育的主要动力”。这就是说，教育的产生完全来自动物的本能，是种族发 展的本能需要。A 项表述不符合生物起源说的观点。

题目:教育具体而实在的规定性表现在（ ）
类型:多选题
A.教育是人类有意识地传递社会经验的活动
B.教育具有经济功能
C.教育是以人的培养为直接目标的社会实践活动
D.教育是人类所特有的一种有意识的社会活动
正确答案:A,C,D

题目:下列关于教育者的说法正确的有( )
类型:多选题
A.教育者是从事教育活动的人，包括学校教师、教育管理人员及参与教育活动的其他人员
B.教育者是教育活动的组织者和领导者
C.教育者是教育过程中“教”的主体
D.教师是最直接的教育者，对受教育者知识、技能、思想、品德等方面的发展起着很大的作用
正确答案:A,B,C,D
解析:教育者是指对受教育者在知识、技能、思想、品德等方面产生教育影响的人， 包括 学校教师、教育管理人员、教科书的设计者和编写者、参与教育活动的其他人员等。 其中，学校教 师是教育者的主体，是最直接的教育者，在教育活动中起着主导作用。教育 者是教育活动中教的 主体，也是教育活动的组织者、设计者和实施者。所以 A、B、C、D 四项均正确。

题目:下列关于教育的内涵，表述正确的观点是( )
类型:多选题
A.“教育”一词最早出自“得天下英才而教育之，三乐也”
B.西方观点认为，教育是“把受教育者内在的东西引导出来”
C. 赫尔巴特认为,教育是经验的改造或改组
D.目的性、计划性和组织性是学校教育具有的特点
正确答案:A,B,D
解析:在我国，一般认为“教育”一词最早见于《孟子 ·尽心上》中的“得天下英 才而教育 之，三乐也”。在西方文化背景下对教育进行词源考察，可以看到， 在西文中，英文、 法文、德文中的“教育”一词均由拉丁文“educare” 演化而来，而拉 丁文“educare” 表示“引出”的意 思。可见，西文中“教育”一词表示把受教育者内在 的东西引导出来。“教育是经验 的改造或改组”是杜威的教育思想。故 C 项 错误。学校教育具有明确的目的性和方向性、较强的 计划性和系统性、高度的组织性。
