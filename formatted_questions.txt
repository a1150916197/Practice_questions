题目: 在我国，"教育"一词最早起源于( )中的"得天下英才而教育之，三乐也"。
类型: 单选题
难度: 中等
选项:
A.《论语》 B.《礼记》 C.《孟子》 D.《说文解字》
正确答案: C
解析: 1.C 【 解 析 】在我国，一般认为"教育"一词最早见于《孟子 ·尽心上》中的"得天下
英才而教育之， 三乐也"。许慎最早在《说文解字》中对"教育"一词进行解释。
易错提示" 教育"一词在我国的最早出处(使用)与最早解释是易混点，考生应注意区分。最
早出处：孟子的《孟子 ·尽心上》———"得天下英才而教育之，三乐也"。
最早解释：许慎的《说文解字》——"教，上所施，下所效也""育，养子使作善也"

题目: 教育活动的基本构成要素包括教育者、受教育者和（ )（易混）
类型: 单选题
难度: 中等
选项:
A.教育影响 B.教育手段 C.教育物资 D.教育活动方式
正确答案: A
解析: 2.A【解 析】由题干及选项设置可知，本题考查的是三要素说。教育者、受教育者(学习者)
和教育 影响(教育媒介)是构成教育活动的基本要素。
方法技巧 关于教育活动的基本要素，不同的学者有不同的观点。以下归纳了几种常考的说
法，考生做题时应注意具体问题具体分析。
三要素说：教育者、受教育者、教育影响/教育媒介/教育措施/教育内容。
四要素说：教育者、受教育者、教育内容、教育手段/教育活动方式/教育物资。

题目: 下列属于教育家孟禄关于教育起源的观点的是（ )
类型: 单选题
难度: 中等
选项:
A.生物的冲动是教育的主要动力
B.教育起源于儿童对成人的无意识模仿
C.教育起源于生产劳动过程中经验的传递
D.教育是在生活中提出的，也是在生活中进行的
正确答案: B
解析: 3.B【 解析】孟禄持心理起源说，他认为教育起源于日常生活中儿童对成人的无意识模仿。
A 项 属 于生物起源说的观点，C 项属于劳动起源说的观点，D 项属于生活起源说的观点。

题目: 教育的本质属性是( )
类型: 单选题
难度: 中等
选项:
A.传授知识 B.传承文化 C.培养人才 D.服务社会
正确答案: C
解析: 4.C【解析】教育的本质属性是育人，即教育是一种有目的地培养人(培养人才)的社会活动。
故 选 C 项。
方法技巧 理解教育的本质属性需要重点抓住"培养人"三个字。(1)"培养人"说明教育活
动不 同于人类的其他社会活动：①它是以人为直接对象的，不同于以物质产品或精神产品
的生产为直 接对象的社会生产活动；②它是以对人的身心发展产生影响为直接目标的，不
同于医疗活动以及 以满足人的需要为目标的社会服务活动。(2)"培养人"确认了教育的存
在特性。教育活动可以 加速人的社会化品质的塑造，通过培养人服务于社会。(3)"培养人"
是教育的主体活动。

题目: 从个体的角度来定义"教育"，往往把"教育"等同于个体的( ）
类型: 单选题
难度: 中等
选项:
A.社会化过程 B.学习与发展过程 C.人格化过程 D.认知过程
正确答案: B
解析: 5.B【解析】一般说来，人们是从两个角度给"教育"下定义的：一个是社会的角度，另一
个是个体 的角度。从个体的角度来定义"教育",往往把"教育"等同于个体学习与发展的
过程。兼顾社会 和个体两个方面给教育下定义：教育是在一定社会背景下发生的促使个体
的社会化和社会的个性化的实践活动。A 项排除，C、D 项为干扰选项。
